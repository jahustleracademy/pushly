# Pushly

Pushly is an iOS-first AI fitness app foundation.

## Scope in this baseline
- Expo Router + TypeScript app shell
- Dark premium theme primitives
- Placeholder screens for core product areas
- Clean infra foundation for Supabase, auth, RevenueCat, analytics
- Feature folders prepared for camera, pose detection, exercise engine, progress, and subscriptions

## Quick start
1. `cp .env.example .env`
2. Fill required keys for Supabase and RevenueCat
3. `npm install`
4. `npm run ios`

## Notes
This repository intentionally excludes legacy recipe/import/nutrition/share-extension flows.
